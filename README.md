Traffic Dataset Cleaner & Analyzer is a smart traffic analytics dashboard that simplifies traffic data preprocessing and exploration. It features:

- Drag-and-drop dataset upload (CSV/XLSX)
- Automated data cleaning and preprocessing
- Interactive KPI dashboard
- Multiple visualization types (Bar, Line, Scatter, Histogram, Pie, Heatmap, etc.)
- Traffic pattern and peak-hour analysis
- Dataset summary and statistical insights
  
  *Designed for students, researchers, and data analysts to quickly transform raw traffic datasets into meaningful insights.*

**Although optimized for traffic datasets, the application is dataset-agnostic and can clean, analyze, and visualize a wide variety of CSV and Excel datasets across different domains.**
